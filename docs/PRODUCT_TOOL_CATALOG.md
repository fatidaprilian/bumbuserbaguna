# Product Tool Catalog (Basic to Advanced)

## 1. Basic Tools
### 1.1 Tugas Check Cepat
1. Input: file tugas atau teks.
2. Output: grammar feedback, readability score, and simple citation hints.
3. User fit: SMP and early SMA.

### 1.2 Struktur Laporan Otomatis
1. Input: type of assignment (laporan praktikum, makalah, proposal).
2. Output: chapter checklist and missing section alerts.
3. User fit: SMP, SMA/SMK.

### 1.3 Plagiarism Basic
1. Input: document upload.
2. Output: similarity percentage and highlighted text overlap.
3. User fit: all levels.

## 2. Intermediate Tools
### 2.1 Plagiarism Advanced
1. Add paraphrase detection and source clustering.
2. Add confidence level per section.
3. User fit: SMA/SMK and university.

### 2.2 Citation Integrity Checker
1. Detect unreferenced quotes.
2. Validate bibliography consistency.
3. Suggest fix by style template.

### 2.3 Presentasi Otomatis dari Laporan
1. Input: final report document.
2. Output: 8 to 15 slide draft with key points, flow, and presenter notes.
3. User fit: SMA/SMK and university.

## 3. Advanced Tools
### 3.1 AI Study Copilot
1. Build study plan from syllabus and deadlines.
2. Generate quiz and flashcards from notes.
3. Track mastery progress.

### 3.2 Feedback Assistant for Teachers
1. Auto draft feedback comments from rubric.
2. Flag class-level plagiarism risk patterns.
3. Provide moderation queue before final release.

### 3.3 Institutional Insights
1. Program-level trends and quality metrics.
2. Cohort integrity analytics.
3. API integration for LMS and SIS.

## 4. Priority Matrix for Implementation
1. Must-have now: plagiarism basic, structure checker, citation helper.
2. Next wave: advanced plagiarism, presentation generator.
3. Later: study copilot, teacher feedback AI, institutional analytics.

## 5. Shared Platform Capabilities
1. Unified document pipeline.
2. Unified AI gateway.
3. Role-based access by education level.
4. Unified billing and quota control.
5. Unified audit and compliance logging.
